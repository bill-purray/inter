## Data_Manager
Data_Manager is a tool that is designed to parse and computate csv data based on the exercises outlined below.

### Exercise
* As a .CSV user I want all States in column “State” to be two characters so that I can enter them in the magic States form. (NY, NJ, NV, etc..) 
* As a .CSV user I want column dogs to be 1 when column cats is 0 so that dogs and cats don’t fight.

### How To
1. Update the interpreter in the execs
2.  `bin/{exec} -p {project}`
